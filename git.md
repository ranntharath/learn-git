# git cammand

```bash
1. git init # initialize git to project
2. git add <file-name> # add files to repository
   git add . # add all files 
3. git commit -m " message " #commit message
4. git branch -M main # create main branch
5. git remote add origin <url> # add repository url
6. git push -u origin main # push code to repository

```
